---

SpecDD agent has expanded beyond the CLI with new agent plugins and IDE support for `.sdd` files.

The agent plugins package the same shared SpecDD skills for multiple coding agents. They help agents read the active
`.specdd/bootstrap.md` chain, resolve local spec authority, and work through orientation, planning, implementation,
review, testing, tracing, documentation, refactoring, debugging, and risk assessment.

Generated agent integrations are now available for Codex, Claude Code, GitHub Copilot, Antigravity, Junie, and Cline.
Each integration is generated from the same source repository, so the workflows stay aligned across agents while still
using each product's native plugin or skill format.

The new JetBrains plugin adds IDE support for SpecDD files, including syntax highlighting, structural validation,
section documentation, structure view entries, section and path completion, clickable project file references,
unresolved path warnings, and create-file quick fixes.

Setup instructions for the CLI, agent plugins, and JetBrains plugin are available on the new [Tools page](/tools/).
